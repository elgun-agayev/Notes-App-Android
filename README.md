# 📝 Notes App Android

A simple and clean Notes application built with **Kotlin**, following **MVVM architecture** and using **Room Database** for offline storage.

---

## 🚀 Features

* ✏️ Create notes
* 📝 Edit notes
* 🗑️ Delete notes
* 🔍 Search notes
* 💾 Offline storage with Room Database
* 🎨 Clean Material Design UI

---

## 🛠️ Tech Stack

* Kotlin
* MVVM Architecture
* Room Database
* RecyclerView
* ViewBinding
* Material Design
